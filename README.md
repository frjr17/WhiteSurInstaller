# WhiteSur Installer

Opinionated WhiteSur setup for Fedora GNOME.

```bash
./install.sh
```

The installer adds the WhiteSur cursor, icon, GTK, libadwaita, GDM, and
wallpaper themes. It also installs Extension Manager from Flathub. Run it as
your desktop user; it asks for `sudo` only when system changes require it.

Other distributions are intentionally rejected until their package and
desktop behavior can be tested.
